dota2protractor.com

This is a web application that I built myself. Dota 2 is a video game that I have played for many years, and the developers have API's available for anyone to use. I incorporated some of those, as well as my own, to create a service that takes a player's ID, and displays information about their most recent game. This project was created with Angular as the front-end and Java SpringBoot as the backend. I hosted the front-end on a Digital Ocean Dropplet, and the backend on AWS Elastic Beanstalk. The database is also an AWS product. All endpoints are SSL encrypted with HTTPS.

Greenpod


Another project worked on with my fellow interns. This one, however, is an actual client project (just without a client- yet!).

RV32I Simulator

This is, by far, my favorite university assignment I have ever worked on (up to this point)! This product uses C++ to simulate a RISCV Processor, as well as memory and registers, which are both represented as vectors.

The program reads a binary file, and translates it into CPU instructions utilizing the simulated registers to move the bits that are in the simulated memory. There are over 50 instructions programmed.

This was a near-semester long project for my Computer Architecture class. It is safe to say that this assignment taught me more about how computers work than anything else I have ever worked on. I hope that my future studies somehow involve the kind of work I did here.
